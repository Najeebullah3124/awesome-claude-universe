# awesome-claude-universe

The best curated collection of Claude Code ecosystem repositories in one place.

## Why this repo exists

This project brings together high-signal Claude Code resources so builders can discover the best tools, examples, plugins, and workflows quickly.

## Top 10 Claude Code Repositories

1. [awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code) - Community mega-list for Claude Code resources
2. [anthropic-cookbook](https://github.com/anthropics/anthropic-cookbook) - Official examples and implementation recipes
3. [repomix](https://github.com/yamadashy/repomix) - Package repositories into AI-friendly context
4. [claude-code-action](https://github.com/anthropics/claude-code-action) - Official Claude Code GitHub Action
5. [awesome-claude-plugins](https://github.com/quemsah/awesome-claude-plugins) - Curated plugin ecosystem list
6. [system-prompts-and-models-of-ai-tools](https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools) - Prompt/system reference collection
7. [anthropic-quickstarts](https://github.com/anthropics/anthropic-quickstarts) - Official quickstart templates
8. [claude-code](https://github.com/anthropics/claude-code) - Official Claude Code CLI
9. [gstack](https://github.com/garrytan/gstack) - Garry Tan's Claude Code setup
10. [ccusage](https://github.com/ryoppippi/ccusage) - Usage analytics and insights for Claude Code

## Publish this repo to GitHub

Run these commands from the `awesome-claude-universe` directory:

```bash
git init
git add .
git commit -m "Create awesome-claude-universe curated Claude Code collection"
gh repo create awesome-claude-universe --public --source=. --remote=origin --push
```

If `gh` is not authenticated yet:

```bash
gh auth login
```
